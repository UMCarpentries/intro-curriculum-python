Python

Python is a popular general-purpose programming language. Installing Python, its scientific packages, and JupyterLab individually can be a bit difficult, so we recommend Anaconda, a Python distribution bundled with other popular tools, including common scientific packages and JupyterLab that we will be using in this workshop.

{% comment %} Please remove or comment out this paragraph using or {% comment %} and {% endcomment %} if you do not plan to use JupyterLab environment. {% endcomment %}

We will teach Python using JupyterLab, a programming environment that runs in a web browser. For this to work you will need a reasonably up-to-date browser. The latest versions of the Chrome, Safari and Firefox browsers are all supported. Note you do not need to install JupyterLab separately as it will come pre-installed with Anaconda.

Note if you have already installed Anaconda previously, you don't need to install it again for this workshop. You do need to make sure your libraries meet the version requirements for this workshop by running the provided `check_setup.ipynb` file. If your libraries do not meet a version requirement, you can run pip install LIBRARY_NAME --upgrade in your command window (i.e., Anaconda Prompt for windows and Terminal for MacOS/Linux) to upgrade the library.

Video Tutorial

  1. Open https://www.anaconda.com/download with your web browser.
  2. Download the Anaconda Installer for Linux.
    (The installation requires using the shell. If you aren't comfortable doing the installation yourself stop here and request help at the workshop.)
  3. Open a terminal window and navigate to the directory where the executable is downloaded (e.g., `cd ~/Downloads`).
  4. Type 
    bash Anaconda3-
    and then press Tab to autocomplete the full file name. The name of file you just downloaded should appear.
  5. Press Enter (or Return depending on your keyboard). You will follow the text-only prompts. To move through the text, press Spacebar. Type yes and press enter to approve the license. Press Enter (or Return) to approve the default location for the files. Type yes and press Enter (or Return) to prepend Anaconda to your PATH (this makes the Anaconda distribution the default Python).
  6. Close the terminal window.
